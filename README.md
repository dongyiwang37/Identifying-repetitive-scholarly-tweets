# Identifying-repetitive-scholarly-tweets
The work is the code of this paper "Repetition and engagement: Analysing the role of repetitive content in original tweets mentioning scientific papers" 

In this work, We only provide the code, which includes sample data in code. Due to licensing and privacy restrictions, we cannot disclose all the data. The complete data for the paper can be found at https://doi.org/10.5281/zenodo.15208246.
